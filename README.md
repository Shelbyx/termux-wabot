## Clone esse git

```bash
> git clone https://github.com/Shelbyx/termux-wabot
```

## Instale as dependências:
Antes de executar o comando abaixo, certifique-se de estar no diretório do projeto que
você acabou de clonar !!

```Comandos para instalar
> cd termux-wabot
> bash install.sh
```

### Comandos para usar
```
> npm start
```

### Prefixo Padrão
```
prefixo = .
```
